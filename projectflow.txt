building pipeline:
1>create a github repo and clone it to local
2>add src folder along with all components
3>add data,model,reports dircts to .gitignore file
4>now git add,commit push

setting up dvc pipeline(without params)
5>create dvc.yaml file and add stages to it
6>dvc init them do dvc repro to test the pipeline automation.(check dvc dag)
7.now git add,commit,push

setting up dvc pipeline(with parms)
8.add params.yaml file
9.add the params setup
10.do "dvc repro" again to test pipeline along with the params
11.now git add,commit,push

exp with dvc
12. pip install dvclive
13.add dvclive code block
14. do "dvc exp run" it will create dvc.yaml and dvclive directory(each run will be considered as an exp)
15. do "dvc exp show" on terminal to see the exp or use extension on vscode
16. do "dvc exp remove{exp-name}" to remove exp(optional) | "dvc exp apply{exp-name}" to reproduce prev exp
17. change params,re-run code
18. git add,commit,push
